# HomeWizard Link app
Old devices, new possibilities! Link your Link to Homey and take your smart home to the next level. 

The app currently supports:
- Color light: on/off, dim, RGB, white temperature
- Door sensor: Contact alarm, tamper alarm

# Device support
Since I only have the Color bulb and the door sensor, I need your help to get the other devices supported. If you have any of the following devices:

- Thermometer
- Wall switch converter
- Siren
- Socket
- Plug-in dimmer
- CO detector

Or any other device that works with the HomeWizard Link app, please let me know in the [forum topic](https://community.homey.app/t/app-pro-homewizard-link/145076) so that I can add support for it.
